# AdventureAlert Help

Alert when a dragon appears in adventure

# .possessedalert
Set notifications for possesseds appearing in adventure<br/>
 - Usage: `.possessedalert`
## .possessedalert global
Toggle possessed notifications across all shared servers<br/>
 - Usage: `.possessedalert global`
## .possessedalert removeuser
Remove a specific user ID from possessed alerts<br/>
 - Usage: `.possessedalert removeuser <user_id>`
 - Restricted to: `MOD`
## .possessedalert add
Toggle possessed notifications on this server<br/>
 - Usage: `.possessedalert add`
 - Aliases: `user, users, remove, and rem`
 - Checks: `server_only`
## .possessedalert role
Add or remove a role to be pinged when a possessed appears<br/>
 - Usage: `.possessedalert role <role>`
 - Restricted to: `MOD`
 - Aliases: `roles`
