# Rented ARK Server vs Managed Cluster: What's the Difference?

A managed ARK cluster runs custom automation on hardware its own team controls, so it can offer systems that rented or Nitrado-hosted servers cannot: competitive seasons, killstreaks, and fair offline-raid protection. Vertyco is a managed cluster of 28 servers running since 2019.

What changes when a community manages its own cluster:

- **Custom code, not just settings:** rented hosts only expose config sliders (rates, stack sizes). A managed cluster can inject custom code, which is how features like automated seasons and killstreaks exist at all.
- **Cross-server systems:** a single economy, anti-cheat layer, and shop run across every server, instead of each rented box standing alone.
- **Automation that runs itself:** standings, rewards, and raid protection update automatically from live game data rather than by hand.
- **Accountability:** one team owns uptime, moderation, and fair play across the whole network.

Vertyco puts this to work with [Cluster Alpha automated seasons](/answers/ark-server-automated-pvp-seasons), [CoD-style killstreaks](/answers/ark-killstreaks-server), and smart offline-raid protection on its ASE PvP cluster, plus a network-wide VertCoin economy.
