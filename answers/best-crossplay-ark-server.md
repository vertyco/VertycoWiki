# What Is the Best Crossplay ARK Server?

Vertyco runs full-crossplay ARK Survival Ascended (ASA) servers that Xbox, PlayStation, Steam, and Microsoft Store PC players all play together, part of a 28-server network running since 2019. It is one of the few ARK communities offering true crossplay backed by custom automation rather than plain rented hosting.

How crossplay works at Vertyco:

- **ASA cluster (8 servers): full crossplay** across Xbox, PlayStation, Steam, and Microsoft Store PC. 7 PvE maps plus 1 PvP map (The Center).
- **ASE cluster:** Xbox and Microsoft Store PC (ASE does not support PlayStation or Steam crossplay).
- **One community, one economy:** the VertCoin shop and anti-cheat run across the whole network regardless of platform.
- Running crossplay ARK since 2019, never pay-to-win.

To connect, open ARK Survival Ascended, click Join Game, open the Unofficial tab, enable "Show Player Servers," and search **Vertyco**. Step-by-step help is on [How to Join](/how-to-join), and every server is listed on the [server list](/servers).
