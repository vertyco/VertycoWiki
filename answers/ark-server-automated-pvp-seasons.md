# Which ARK Server Has Automated PvP Seasons?

Vertyco's ASE PvP cluster runs Cluster Alpha, an automated competitive season system that crowns the best tribe from real PvP and build data, with no admin bias and no favoritism. No other ARK community automatically ranks and rewards its top tribe from live data like this.

How Cluster Alpha's automated season works:

- **Live Power rating:** every tribe earns a rating from what it actually builds.
- **Zero-sum Alpha Points:** points are stolen through PvP, never handed out. Kill or raid an enemy and their points transfer to you.
- **Kills pay out:** rewards scale with the victim's power, with underdog bonuses, bounties on top tribes, and "most wanted" bonuses for hunting top killers.
- **Killstreak medals, rivalries, and automatic daily and weekly recaps** of the standings.
- **Season rewards:** the top 3 tribes get VertCoin payouts before each wipe, and 1st place earns a starter pack for next season.

Cluster Alpha is a custom code-injected system exclusive to the ASE PvP cluster (not possible on rented, Nitrado, or ASA servers).
