# Autto Bot Commands

This page provides documentation for Autto's features and commands. Click a link below to learn more about a specific plugin.

## 🎮 Games & Minigames
- 🎰 [**Casino**](/cogs/casino): Gamble your VC and win big with slots and other games.
- 🔢 [**Counting**](/cogs/counting): A counting channel game for the server.
- 🏹 [**Hunting**](/cogs/hunting): Hunt birds and other flying things for rewards.
- ⛏️ [**Miner**](/cogs/miner): Mine resources and trade with other players.
- 🖼️ [**Pixl**](/cogs/pixl): An image-guessing game with points.
- 🐶 [**Pupper**](/cogs/pupper): Pet the doggo!
- 🧠 [**Trivia**](/cogs/trivia): Play trivia with your friends!

## 💰 Economy & Rewards
- 💵 [**Economy**](/cogs/economy): Get rich and have fun with virtual currency!
- 📊 [**EconomyTrack**](/cogs/economytrack): View economy balance graphs over time.
- 📈 [**LevelUp**](/cogs/levelup): A leveling system with experience, profiles, and leaderboards.
- 🔗 [**Referrals**](/cogs/referrals): Refer friends and earn rewards.

## 🛠️ Ark Server Tools
- 🦖 [**ArkTools**](/cogs/arktools): Player commands for Ark servers - register, buy items from Rshop, and more.

## 🤖 Utility
- 💬 [**Assistant**](/cogs/assistant): Your personal AI assistant - chat, generate images, and more.
- 📅 [**Events**](/cogs/events): Join server events and submit entries.
- 🌐 [**Fluent**](/cogs/fluent): Seamlessly translate messages between languages.
- 🎁 [**Giveaways**](/cogs/giveaways): Enter giveaways hosted in the server.
- 💡 [**IdeaBoard**](/cogs/ideaboard): Share your ideas and suggestions.
- ⏰ [**Reminders**](/cogs/reminders): Set reminders so you don't forget anything.
- 🎟️ [**Tickets**](/cogs/tickets): Open support tickets to get help from staff.
- 🔧 [**XTools**](/cogs/xtools): View Xbox profiles and gamertag information.

## 🛡️ Moderation Tools
- 😀 [**EmojiTracker**](/cogs/emojitracker): Track emoji usage statistics in the server.
- 🔨 [**VrtUtils**](/cogs/vrtutils): Various bot utilities and information commands.
