# Ark Data Help

This section will help with guiding players that are having Ark data problems

# Helping With Ark Data Issues

Keep in mind that players often use different terminology for things like obelisk/terminal/upload ect... So make sure you understand what they're referring to.

# Player Can't See Their Data

If a player knows they should have something in their Ark data but cannot see it, you can use the .arkdata info command to give a short troubleshoot guide for resolving this bug. If none of those steps work, proceed below.
