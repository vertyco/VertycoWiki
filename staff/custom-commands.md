The following are custom commands created by our staff members over the years and are used to help players by providing snippets of info rather than having to type it out every time.
- `.xboxnat` - Get help with Xbox NAT issues
- `.win10nat` - Get help with Windows 10 NAT issues (sends a GIF)
  - `.win10nat2` - Sends a link for troubleshooting windows 10 NAT issues
- `.clearcache` - Get help clearing Xbox Cache
- `.teredo` - Get help with Xbox Teredo issues
- `.decaytime` - Shows current decay times for different structures
- `.joinasa` - Sends an image showing how to join the ASA servers
- `.arkchat` - Sends a link to the wiki showing how to use in-game chat
- `.wiki` - Sends a link to a wiki page of ALL available bot commands
- `.waveform` - Sends a link to run a speed test on your PC and check for buffer bloat
- `.privacy` - Sends a link to Xbox privacy settings for when players can't get the map to add them

# Viewing CustomCommands
Type `.cc list` to view all of the available custom info commands
